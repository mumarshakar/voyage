Body Text Example

▼

This is a paragraph of body text demonstrating the default typography settings. The font is Segoe UI at 16px with regular weight (400). The line height is set to 1.6, which provides comfortable reading spacing. Letter spacing is 0px, and the color is #333333, ensuring good readability.

Multiple paragraphs maintain consistent styling. This helps create a cohesive reading experience throughout your website. The typography settings can be customized through the Shopify theme editor.
