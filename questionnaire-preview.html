Technical Notes

Features
  • All fields are required (HTML5 validation)
  • JavaScript validation before form submission
  • Data stored as Shopify cart item properties
  • Properties display on cart page automatically
  • Properties included in order confirmation
  • Responsive design (mobile-friendly)
  • First Name/Last Name excluded (collected at checkout)
Important Notes
  • Properties persist through cart updates
  • If item removed and re-added, form must be filled again
  • Properties are read-only on cart page
  • Order page automatically includes properties (no code needed)
  • Max 100 properties per cart item (we use 5)
  • Property value max 5,000 characters
